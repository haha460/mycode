
校验 & 数据库连接池
===============
数据库连接池

> * 单例模式，保证唯一
> * list实现连接池，作为连接的容器
> * 连接池为静态大小
> * 互斥锁实现线程安全
> * 获取连接和归还连接时使用信号量sem实现同步
> * RAII 实现超出作用域自动归还连接

* 数据库连接：
声明：MYSQL* con
初始化：con = mysql_init(con)
连接：mysql_real_connect(con, ...);
释放连接：mysql_close(con)

校验  
> * HTTP请求采用POST方式
> * 登录用户名和密码校验
> * 用户注册及多线程注册安全
